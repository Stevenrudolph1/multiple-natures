# Multiple Natures

MN theory, MNTEST assessment tool, practitioner network, and legacy MNI materials. MN is now the Alignment lens within Renergence — this project holds the domain-specific source material.

## Directory Structure

```
multiple-natures/
├── theory/                    <- MN framework, canon docs, Alignment Dynamics
│   ├── canon-docs/            <- 9 natures, 10 intelligences, brand, FAQ, design
│   ├── ad-theory/             <- Alignment Dynamics framework + legacy docs
│   └── tag-architecture/      <- TAG architecture canon
├── mntest/                    <- Assessment instrument
│   ├── architecture/          <- Agent architecture, trait model, scoring model
│   ├── question-bank/         <- Question CSV, brackets, item bank
│   ├── reports/               <- Sample reports, session data
│   ├── career-matching/       <- Future: career profile matching data
│   └── research/              <- Future: validation studies, psychometric data
├── legal/                     <- MNI corporate, settlements, contracts
│   ├── kalba/                 <- Settlement docs (Rytis/EduPerfectus)
│   ├── sussman/               <- Publisher certificate
│   └── practitioner-agreement/
├── practitioners/             <- Network and communications
│   ├── affiliation-program/   <- MN-AFF docs (001-015) + mini website
│   └── comms/                 <- Emails, newsletters, video messages
├── governance/                <- MN-GOV docs (00, 01, 10, 11, 20)
├── courses/                   <- Seeing Course materials (EN/FR)
├── website/                   <- multiplenatures.com static site
├── _archive/                  <- Deprecated loose files
└── README.md
```
